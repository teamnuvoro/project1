# Image Assets for Riya AI

## How to Update the Landing Page Image

### Option 1: Replace the downloaded image
1. Save your desired image as `riya-main.jpg` in this folder
2. Make sure it's high quality (at least 1200px wide)
3. The app will automatically use it

### Option 2: Use the provided image
If you have the woman in red kurta image:
1. Save it as `riya-main.jpg` in `/client/public/images/`
2. Restart the dev server
3. The landing page will show your image
