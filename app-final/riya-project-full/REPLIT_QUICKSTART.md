## One-Time Setup

1. **Import from GitHub** (Easiest)
   - Click "Import from GitHub" in Replit
   - Enter: `teamnuvoro/project1`
   - Click Import

2. **Or Clone Manually**
   ```bash
   cd /home/runner/workspace
   git clone https://github.com/teamnuvoro/project1.git .
   ```

## Daily Use

### Pull Latest Code (When you push from laptop)
```bash
bash replit-git-sync.sh pull
```

### Push Your Changes (After editing in Replit)
```bash
bash replit-git-sync.sh push
```

### Full Sync (Pull + Push)
```bash
bash replit-git-sync.sh sync
```

### Super Quick Sync (One command)
```bash
bash replit-quick-sync.sh
```
