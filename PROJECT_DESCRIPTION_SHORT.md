# Riya AI Companion - Brief Description

**Riya** is a production-ready AI-powered relationship companion platform featuring natural Hinglish conversations, real-time voice calling, and personalized relationship insights. Built with React/TypeScript, Node.js/Express, PostgreSQL, and integrated with multiple AI services (Groq, OpenAI, Sarvam AI, ElevenLabs).

**Key Features:**
- 4 distinct AI personas with evolving personalities based on user interactions
- Real-time voice conversations via WebSocket with low-latency audio processing
- Advanced memory system with relationship depth tracking and vector search
- Streaming chat responses with Server-Sent Events
- Complete payment/subscription system (Cashfree/Razorpay)
- WhatsApp-like mobile-first UI with dark mode
- AI-generated relationship insights and analytics

**Tech Stack:** React 18, TypeScript, Vite, TailwindCSS, Express.js, PostgreSQL (Drizzle ORM), WebSocket, SSE, Groq API, OpenAI API, Sarvam AI, ElevenLabs, Cashfree, Razorpay

**Demonstrates expertise in:** Full-stack development, AI/ML integration, real-time systems, payment processing, modern React patterns, TypeScript, database design, and production deployment.
